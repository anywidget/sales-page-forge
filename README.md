# Sales Page Forge

A micro-SaaS visual page builder for creating Internet Marketing-style sales letters and JV (Joint Venture) pages.

## Features

- **Visual Drag-and-Drop Editor** - Build high-converting sales pages with an intuitive section-based editor
- **12 Professional Themes** - Choose from Internet Marketing-inspired themes like Classic Red, Money Green, Dark Authority, and more
- **Text Effects System** - Add eye-catching effects including neon glow, text shadows, outlines, and dramatic letter spacing
- **Freeform Elements** - Position text boxes and images anywhere on your page with the hybrid editor
- **AI Copy Writer** - Generate persuasive sales copy using OpenAI, Claude, or Gemini (bring your own API keys)
- **Stock Image Search** - Find perfect images via Pexels, Unsplash, or Pixabay integration
- **HTML Export** - Download your finished pages as standalone HTML files
- **Tiered Pricing** - Free tier for testing, Monthly and Lifetime plans for full access

## Section Types

- Hero sections with background images and CTAs
- Subheadlines and body copy
- Bullet point lists
- Testimonials with avatars
- FAQ accordions
- Video embeds
- Bonus stacks
- Money-back guarantees
- JV-specific sections (commission tables, prize calendars)

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Express.js, Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: Session-based with bcrypt password hashing

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Set up your PostgreSQL database
4. Configure environment variables
5. Run the development server: `npm run dev`
